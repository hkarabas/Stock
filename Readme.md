This project for payconiq task of hiring

--Spring Boot 2.5.6

--liquibase for Db migration and initial Data load 

--MySqlDb Relation Db

--Docker deployment,build and Run together with  Docker Compose 

--Swagger Api Documention.

Run with Docker:
